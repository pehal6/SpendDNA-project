# 💳 SpendDNA – Personal Transaction Analysis

🚀 *Mini Project 2 | Unlox Internship*

---

## 📌 Overview

SpendDNA is a data analysis project focused on understanding personal financial behavior using real-world transaction data.

The goal of this project is not just to clean data, but to extract meaningful insights about spending patterns, lifestyle habits, and financial trends over time.

---

## 🎯 Objectives

* Clean and preprocess raw bank transaction data
* Standardize inconsistent formats (dates, amounts, transaction types)
* Identify and group vendors intelligently
* Categorize expenses into meaningful buckets
* Analyze spending trends and financial behavior
* Generate insights that actually tell a story

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **NumPy**
* Jupyter Notebook

---

## 📊 Key Features

### 🔹 Data Cleaning & Preparation

* Handled messy date formats using robust parsing
* Cleaned currency values (₹, commas, text noise)
* Standardized credit/debit formats
* Removed duplicates and invalid entries

---

### 🔹 Financial Analysis

* Total Credits vs Debits
* Category-wise spending distribution
* Top vendors by spend
* Percentage contribution of each category

---

## 🧠 What I Learned

* Real-world data is messy — cleaning takes more effort than analysis
* Small preprocessing mistakes can completely distort insights
* Categorization is the most critical step in financial analysis
* Writing simple, readable code > over-complicating logic
